# Раздели сметката

Приложение за разделяне на ресторантска сметка между приятели. Работи изцяло в браузъра — снимките не се изпращат на сървър.

## Как да го качиш в GitHub

1. Създай нов репозиторий в GitHub (например `SplitBill`).
2. Качи тези файлове в корена на репозитория:
   - `index.html`
   - `styles.css`
   - `app.js`
3. В репозитория отиди на **Settings → Pages**.
4. Под **Source** избери **Deploy from a branch**.
5. Избери клон **main** (или **master**) и папка **/ (root)**. Запази.
6. След минута-две приложението ще е на адрес:  
   `https://<твоето-потребителско-име>.github.io/SplitBill/`

## Как се ползва

1. **Качи снимка** на касовата бележка (или направи снимка).
2. Натисни **Извлечи текста от снимката** (за автоматично попълване) или **Въведи ръчно**.
3. **Редактирай редовете** — добави/премахни артикули и цени, така че да отговарят на бележката.
4. **Добави имената** на всички, които споделят сметката.
5. **Разпредели** всеки ред към един или няколко човека (чекбоксове).
6. **Виж обобщението** — колко дължи всеки.

Цената на всеки ред се разделя по равно между избраните за него хора.
